# Desktop Host (Tauri)

Нативное приложение: ввод URL внешнего API, предпросмотр, локальный **gateway** для OBS (`/api/state`, `/ws`, статика оверлея).

## Кратко

- После **Запуск сервера**: `http://127.0.0.1:<порт>/` — оверлей; **Тест** — тот же сервер без опроса внешнего API (порт по умолчанию **8787**).
- Разработка из этого каталога: `npm install` → `npm run tauri:dev`.
- Установщик: `npm run tauri:build`.

**Полная документация:** [../../docs/README.md](../../docs/README.md) (разработка, архитектура, внешний API, CI).
